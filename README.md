# Hangman Series

Hangman Series is an interactive web-based game built using HTML, CSS, and JavaScript. Players can enjoy multiple Hangman games across different themes while improving their vocabulary and problem-solving skills.

## Features
- Browser-based gameplay
- Interactive interface
- Multiple Hangman categories
- Built with HTML, CSS, and JavaScript
- No installation required

## How to Play
1. Open the webpage.
2. Select a Hangman game.
3. Guess one letter at a time.
4. Complete the word before running out of guesses.

## Technologies Used
- HTML5
- CSS3
- JavaScript
